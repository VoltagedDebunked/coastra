# Coastra
 Coastra is a Widget and GUI library made in Rust to build modern applications with Rust.
## Installation

Run:

```bash
cargo add coastra
```

To add Coastra to your project.

## Usage

Go to `docs.rs/coastra` to see the Documentation.
